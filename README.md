How to get the app running:

////

**Quick check: Make sure you have python 3.11 installed onto your computer before getting started on this**

////

**Consumer Settings**

**Potato Mode
CPU / Chromebook-ish / old laptop
FP32, low batch, 128–192 resolution**

**Laptop Safe
FP32 or Auto, small batch, low workers**

**Normal GPU
Auto precision, medium batch**

**RTX 3060 12GB
FP16/Auto, batch 32, strong prefetch/cache**

Monster GPU
Higher batch, high cache, high resolution

////

Step 1: extract the zip file to access the folder inside.

step 2: right click on the background and open up the terminal

step 3: type in this into the terminal "Pip install -r requirements.txt"

step 4: double click on the app.py file to see if it works

step 5: Enjoy the app
